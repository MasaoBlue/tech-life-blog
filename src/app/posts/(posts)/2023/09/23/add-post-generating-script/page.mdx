正直ブログとして開設しておきながら、Zenn とどう使い分けようとか ZennZenn 考えてなかった。

多分 React で何か試して、記事の中で動くものを置いて置きたい時とかに使うんだろうなと思ってはいる。

とりあえず、いざ書き始めようとした時のハードルを少しでも下げるために、記事の mdx を生成するスクリプトを追加した。

## 使い方

```sh
./create-today-post.sh my-post-slug
```
